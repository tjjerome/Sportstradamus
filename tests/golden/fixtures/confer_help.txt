Usage: confer [OPTIONS]

  Fetch current odds and player props into the archive.

Options:
  --close-lines            Targeted close-line scrape only. Reads
                           data/upcoming_events.json and hits per-event
                           endpoints for games starting within the closing
                           window. Exits without touching archive or API when no
                           events are due.
  --fixture-dir DIRECTORY  Read canned Odds API responses from this directory
                           instead of hitting the live API. Used by the
                           integration test suite. Skips game-level moneyline
                           fetches; player props are loaded from
                           <fixture_dir>/sports.json,
                           <fixture_dir>/events_<sport_key>.json, and
                           <fixture_dir>/odds_<sport_key>_<event_id>.json.
  --help                   Show this message and exit.
